# 访问控制系统项目完成说明

## 项目概述

本项目是一个完整的访问控制系统原型，为BigComp公司总部设计。系统实现了基于徽章的访问控制，包括用户管理、徽章管理、资源管理、配置文件管理、日志记录和实时监控等功能。

## 已完成的功能

### ✅ 核心功能
1. **用户管理**
   - 用户注册、更新、删除
   - 支持多种用户类型（员工、承包商、实习生、访客）
   - 完整的GUI界面

2. **徽章管理**
   - 徽章创建、分配、注销
   - 徽章有效期管理
   - 徽章与用户关联

3. **资源管理**
   - 资源信息管理
   - 资源类型分类（门、电梯、打印机等）
   - 资源状态控制

4. **配置文件管理**
   - 配置文件创建和管理
   - 配置文件与资源组关联
   - 配置文件与徽章关联

5. **访问控制**
   - 访问请求处理
   - 徽章验证
   - 权限检查
   - 访问规则验证（时间、日期等）

6. **日志系统**
   - CSV格式日志记录
   - 按年/月/日组织日志文件
   - 日志查询和过滤功能

7. **实时监控**
   - 站点平面图可视化
   - 楼层平面图可视化
   - 实时访问请求显示
   - 活动日志记录

### ✅ 技术实现
1. **架构设计**
   - 分层架构（Model-DAO-Service-GUI）
   - 接口与实现分离
   - 单例模式（数据库连接、日志管理）
   - DAO模式

2. **数据库设计**
   - 完整的数据库表结构
   - 外键约束
   - 索引优化
   - 示例数据

3. **GUI设计**
   - Java Swing界面
   - 标签页组织
   - 表格显示
   - 表单输入

## 项目结构

```
project/
├── src/main/java/com/bigcomp/accesscontrol/
│   ├── Main.java                    # 主入口
│   ├── model/                       # 实体类（8个）
│   ├── dao/                         # 数据访问层（8个接口+8个实现）
│   ├── service/                     # 业务逻辑层（1个接口+1个实现）
│   ├── gui/                         # GUI界面（7个类）
│   └── util/                        # 工具类（2个）
├── src/main/resources/config/      # 配置文件
├── database/                        # 数据库脚本
├── files/                           # 配置文件和资源文件
├── README.txt                       # 安装运行说明
├── pom.xml                          # Maven配置
└── build scripts                    # 构建脚本
```

## 构建和运行

### 构建项目
```bash
# Windows
build.bat

# Linux/Mac
chmod +x build.sh
./build.sh
```

### 运行项目
```bash
java -jar access-control-system.jar
```

## 注意事项

1. **数据库连接**：确保MySQL服务运行，并正确配置database.properties
2. **日志目录**：系统会自动创建logs目录，确保有写入权限
3. **配置文件**：配置文件存储在files目录，需要确保目录存在
4. **图片资源**：如果需要显示站点和楼层平面图，需要添加相应的图片文件到resources目录
